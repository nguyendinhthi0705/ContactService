# ContactService
Demo .NET 6 Azure Function and Cosmos Database.

#To run the app, 
1. Create CosmosDB  
2. Add Appsettings.json with the following key:COSMOS_DB_CONNECTION_STRING  
3. Use the file SampleContact.json for sample data to add a new contact
